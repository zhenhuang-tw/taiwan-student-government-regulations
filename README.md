# 臺灣學生自治規章彙編

整理、瀏覽各級學校學生自治組織自訂規章的線上平台。

## 專案目標

* 提供學生自治參與者便捷的規章查詢工具。
* 提供規章歷史存檔，供研究。

## 技術棧

* **前端框架**: Nuxt 4 (Vue, Vite)
* **程式語言**: TypeScript
* **CSS 框架**: Tailwind CSS
* **資料儲存/CMS**: Nuxt Content V3
* **版本控制**: Git / GitHub
* **部署**: Cloudflare Pages (SSG 靜態部署)

## 功能概覽

* **首頁**: 列出所有學生自治組織，並顯示該組織已收錄的各規章。
* **規章瀏覽**: 預設瀏覽規章的最新版本，亦支援歷史版本追溯，可輕鬆切換查看規章的各個修訂版本。
* **暗色模式**: 提供舒適的夜間閱讀體驗。
* **章節架構**: 側邊提供章節架構概覽。

## 開發設置

依賴套件使用 pnpm 管理。

## 資料格式

### `content/regs/:org/:regSlug/:yyyy-mm-dd.md`

純文字文件，頂部包含 YAML Front Matter。

請注意相同規章、不同修訂版本 belongsTo, versionOf 兩個值要統一，系統才會將之辨識為同一規章。

```md
---
fullTitle: 國立虎尾高級中學學生會組織章程
shortTitle: 組織章程
note: 2022年2月20日學生代表大會修訂通過
isCurrent: true
source: 學生會網站
belongsTo: 虎尾高中學生會
versionOf: 組織章程
version: 2022-10-14
---

## 第一章 總則

第一條

國立虎尾高級中學學生會（以下簡稱本會）為本校最高學生自治團體...
...
```

## 著作權

各學生自治組織法規之著作權，仍歸其所有。
